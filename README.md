# wordpress-dummy-post-vue

This APP is developed in Vue 3 to display wordpress Posts.

Wordpress URL:
https://eowparis.wordpress.com/

Vue app with following structure

- **router**
- **services to fetch data using WP public API**
- **environment variable**
- **pages and component structure**

To Install node packages <br />
```cd <folder-name>```<br />
``` npm install ```

To Run Application<br />
``` npm run serve ```
